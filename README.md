# 🏈 NFL Draft Classifier

A simple machine learning project that predicts whether a college football athlete is likely to be selected in the NFL Draft.

The project was built for the **NFL Draft Prediction** competition. It uses player measurements, athletic testing results, school, and position information to generate draft probability predictions.

## 🎯 Project Goal

The goal is to predict the `Drafted` column:

- `1` means the player was drafted.
- `0` means the player was not drafted.

For the test data, the model outputs a probability between `0` and `1`.

Example:

```csv
Id,Drafted
2781,0.475494
2782,0.773260
```

## 📊 Evaluation

The competition uses **AUC** as the scoring metric.

AUC measures how well the model ranks drafted players above non-drafted players. A higher score is better.

## 🧠 Model Approach

The improved pipeline:

- Uses only the provided competition data.
- Adds useful features like BMI, speed, power, jumping, and agility ratios.
- Handles missing values with median imputation.
- Encodes categorical columns like `School`, `Position`, and `Player_Type`.
- Trains models with 5-fold cross-validation.
- Creates a valid `submission.csv` file automatically.

The strongest public leaderboard score so far is **0.84999**, from a CatBoost submission with position-relative athletic features.

New stronger candidates were also generated with **XGBoost** and **CatBoost**. They need leaderboard submission to confirm whether they beat `0.8365`.

## 🚀 How To Run

Create and activate a virtual environment:

```bash
python3 -m venv .venv
source .venv/bin/activate
```

Install dependencies:

```bash
python -m pip install --upgrade pip
python -m pip install -r requirements.txt -e .
```

Train the model and generate a submission:

```bash
python -m nfl_draft_classifier.train --model gradient_boosting
```

Recreate the best next candidate submission:

```bash
python -m nfl_draft_classifier.train --model gradient_boosting --model-seed 2025 --split-seed 99
```
